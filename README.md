## 🎯 About This Repository

This repository is part of the **Capgemini Frontend with Angular** training program. Each folder corresponds to a specific day of training, containing practice files, exercises, and examples explored during that session.

The goal is to build a solid foundation in frontend technologies, progressing from HTML and CSS basics through JavaScript, TypeScript, and finally to Angular — a powerful framework for building scalable, dynamic web applications.

## 🚀 Getting Started

To explore the code locally:

```bash
# Clone the repository
git clone https://github.com/cser-naivedya-raj/Capgemini-Frontend-With-Angular.git

# Navigate into the project folder
cd Capgemini-Frontend-With-Angular

# For HTML, CSS, JS projects: Open any HTML file in your browser to view it
# For TypeScript projects: Run `tsc` in the project folder to compile
# For Angular projects: Run `npm install` then `ng serve` in the project folder
```

HTML, CSS, and JavaScript projects require only a web browser. TypeScript and Angular projects require Node.js and npm. 🌐
