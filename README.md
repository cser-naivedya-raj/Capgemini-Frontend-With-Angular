## 🎯 About This Repository

This repository is part of the **Capgemini Frontend with Angular** training program. Each folder corresponds to a specific day of training, containing practice files, exercises, and examples explored during that session.

The goal is to build a solid foundation in frontend technologies before moving on to Angular — a powerful framework for building scalable, dynamic web applications.

## 🚀 Getting Started

To explore the code locally:

```bash
# Clone the repository
git clone https://github.com/cser-naivedya-raj/Capgemini-Frontend-With-Angular.git

# Navigate into the project folder
cd Capgemini-Frontend-With-Angular

# Open any HTML file in your browser to view it
```

No build tools or dependencies are required — just a web browser! 🌐
